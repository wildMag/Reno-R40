La pluspart des styles expliqués ci-dessous sont définis par défaut (forms.css) pour l'entièreté du site. Lors de développements spécifiques il est donc conseillé de ne pas changer les styles par défaut mais de redéfinir ceux qui s'appliques aux nouveaux développements dans le fichier custom.css. Pour faciliter la personnalisation, nous vous conseillons l'utilisation d'un conteneur avec id ou classe spécifique.

Définir un formulaire contenant des fieldsets

La balise <fieldset> permet de regrouper des champs et informations de même thème ou de même nature. Chaque <fieldset> doit obligatoirement contenir une balise <legend> qui définit la légende du regroupement.

Code XHTML

            <div class="searchForm">
                <form action="" method="post">
                  <fieldset>
                  <legend>Votre recherche</legend>
                    <div class="formContent">
                    </div>
                  </fieldset>
                  <fieldset>
                  <legend>Date de parution</legend>
                    <div class="formContent">
                    </div>
                  </fieldset>
                </form>
            </div>

Code CSS

Classes CSS
Classe Utilisation

 Conteneur permettant d'identifier le formulaire afin de lui donner des styles spécifiques 
fieldset

legend
 Couleurs, marges et position de la légende 
.formContent
 Couleurs, marges et bordures de la zone de contenu
